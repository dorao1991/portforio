PROFILE

プロフィール

Tokonami Hiroki

エンジニアとして働きながらWeb制作に挑戦

1991年生まれの30歳。 2016年に通信機器メーカーに入社後エンジニアとして勤務を開始し、主に通信機器を操作する為のWeb周りの開発を担当。 2020年頃、趣味であるサーフィン繋がりから知り合いのサーフショップのWebサイト制作を頼まれたことをきっかけに、2021年頃から本格的にWeb制作の活動を行なう。 エンジニア職で培ったコーディング力やWeb技術の知見を活かしつつ、デザインにも挑戦し幅広いスキルを持ったエンジニアを目指す。

詳しい経歴について

TIMELINE

  • 2016 / 04

    移動体通信技術向けの通信計測機メーカーに入社

  • 2016/05

    未経験から研修を通して初めてプログラミング(C言語)に触れる

  • 2017/08

    通信機器の開発プロジェクトにアサインされる。主にAPI周りを担当。

  • 2017/09〜2020/09

    通信機器のバージョンアップにおける、機能追加/不具合対応を担当

  • 2020/09

    通信機器のリプレイス開発において、API設計/実装を担当

  • 2021/05

    副業としてWeb制作を開始

  • 2021/06

    通信機器のGUI改修を一部担当。本業でもフロントエンド開発に携わるようになる

  • 2021/11

    Web制作にて初案件を受注

  • 2022/05

    Web制作/Web開発をメインとした会社へ転職活動を決意
